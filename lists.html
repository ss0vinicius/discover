How to organize your code?

Simple steps to make it easier to ready

  1. First, you need to understand what you will code;
  2. Then, you need to create the main structure of your code;
  3. Next, it is very important to organize your code rows appling indent;
  4. Another important thing is to use names to your variables, classes and elements that are easy to understand why it exists;
  5. If you have some code file with a lot of rows, probably it is better to break it into smaller pieces;
  6. An interesting tip is to use only one language in all the code, as we already use English as commands to our codes, it is interesting to use only English as the name of functins, classes, variables...
  • If you thing it is better to keep doing it exactly how it is today, keep doing.
  • But if you think you can improve your code, we can help you.
